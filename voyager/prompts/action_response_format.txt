You MUST output a JSON dictionary with the following structure:

{
  "program_code": "<async function solving ONLY the top-level task>",
  "program_name": "<functionNameInCamelCase>",
  "reasoning": "<brief explanation>",
  "recipe": {
    "inputs": ["<list of items required>"],
    "output": "<item produced by this skill>",
    "craftIn": "<crafting_table | furnace | 2x2_grid | null>"
  }
}

===============================
CONSTRAINTS FOR program_code
===============================

- You MUST produce exactly ONE async function.
- The function MUST take (bot) as its only parameter.
- The function MUST contain DIRECT primitive calls, for example:

    await mineBlock(bot, 'oak_log', 2);
    await craftItem(bot, 'oak_planks', 4);

- You MUST NOT create ANY new helper functions for gathering or mining.
- You MUST NOT define new skills whose only purpose is gathering.
- You MUST NOT include inventory checks — the system resolves these.

===============================
EXAMPLES (STRICTLY FOLLOW THESE)
===============================

Example for crafting with direct primitives:

{
  "program_code": "async function craftWoodenPickaxe(bot) {\n  bot.chat('Crafting wooden pickaxe');\n  await mineBlock(bot, 'oak_log', 1);\n  await craftItem(bot, 'oak_planks', 4);\n  await craftItem(bot, 'stick', 2);\n  await craftItem(bot, 'wooden_pickaxe', 1);\n  bot.chat('Pickaxe crafted!');\n}",
  "program_name": "craftWoodenPickaxe",
  "reasoning": "Use primitives directly. No helper skills.",
  "recipe": {
    "inputs": ["oak_log", "oak_planks", "stick"],
    "output": "wooden_pickaxe",
    "craftIn": "crafting_table"
  }
}

You may return immediate primitive execution using ONLY the field:

    "exec_code": "await mineBlock(bot, 'oak_log', 1);"

In this case:
- program_code MUST be empty string ""
- program_name MUST be empty string ""
- The system will run exec_code immediately and will NOT store a new skill.

Example for gathering inside main function (NOT a standalone skill):

{
  "program_code": "async function gatherStoneForFurnace(bot) {\n  bot.chat('Mining stone');\n  await mineBlock(bot, 'stone', 8);\n  bot.chat('Stone ready');\n}",
  "program_name": "gatherStoneForFurnace",
  "reasoning": "Direct primitive use.",
  "recipe": {
    "inputs": ["stone"],
    "output": "stone",
    "craftIn": null
  }
}
